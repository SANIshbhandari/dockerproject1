FONT AWESOME OFFLINE SETUP INSTRUCTIONS
========================================

To make Font Awesome work offline, follow these steps:

STEP 1: Download Font Awesome
------------------------------
1. Go to: https://fontawesome.com/download
2. Click "Free For Web" download button
3. Extract the downloaded ZIP file

STEP 2: Copy Files
------------------
From the extracted folder, copy these folders to this directory (FarmSatthi/assets/fontawesome/):
- css/
- webfonts/

Your structure should look like:
FarmSatthi/
  assets/
    fontawesome/
      css/
        all.min.css
        fontawesome.min.css
        (other css files)
      webfonts/
        fa-solid-900.woff2
        fa-regular-400.woff2
        fa-brands-400.woff2
        (other font files)

STEP 3: Update home.php
------------------------
Replace this line:
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.0.0/css/all.min.css">

With this:
<link rel="stylesheet" href="assets/fontawesome/css/all.min.css">

STEP 4: Test
------------
1. Disconnect from internet
2. Open your website
3. Icons should still display

ALTERNATIVE: Use the CDN (requires internet)
---------------------------------------------
If you want to keep using CDN (current setup), it will work fine when internet is available.
The CDN is faster and doesn't require local storage.

For production/defense:
- If presenting with internet: Keep CDN (current setup is fine)
- If presenting without internet: Follow steps above
